# Accessor method for objects from Intensity Analysis

Accessor method for objects from Intensity Analysis

## Usage

``` r
# S4 method for class 'Interval'
x$name

# S4 method for class 'Category'
x$name

# S4 method for class 'Transition'
x$name
```

## Arguments

- x:

  the object

- name:

  the name of the object

- Interval:

  the class

- Category:

  the class

- Transition:

  the class
